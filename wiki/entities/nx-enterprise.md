# Nx Enterprise

## Overview

Nx Enterprise is an enterprise support program offered by the Nx team for companies implementing monorepo architectures. The program provides guidance, best practices, and support based on experience working with Fortune 500 companies.

## Services

### Expert Guidance
- Workspace organization consulting
- Architecture review and recommendations
- Migration planning and support
- Performance optimization

### Best Practices
The Nx Enterprise program has developed best practices through work with numerous large organizations, including:
- [[Modulith Architecture]] patterns
- [[Workspace Structure Goals]] frameworks
- [[Module Boundary Rules]] enforcement strategies
- Team structure alignment with code organization
